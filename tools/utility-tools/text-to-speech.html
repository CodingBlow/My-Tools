Frequently Asked Questions

What is text-to-speech?
Text-to-speech (TTS) converts written text into spoken audio using synthetic or neural voices, useful for accessibility, education, and content creation.
Is the TTS tool secure?
Yes, standard mode processes everything in your browser. Premium voices use server-side Google Cloud TTS, so ensure you trust the provider.
Which browsers support premium voices?
Chrome and Edge often include high-quality voices. Voice availability varies by browser and OS.
How do I use premium voices?
Enable the premium voices option and provide a Google Cloud TTS API key. This requires an internet connection and server-side processing.
Are there limits on text length?
For best performance, keep text under 5000 characters, as longer texts may be truncated by some browsers.
